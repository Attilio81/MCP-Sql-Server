🔒 SECURITY
   • SQL injection prevention with prepared statements
   • Table blacklist with wildcard support (sys_*, *_audit, etc.)
   • Schema whitelist for fine-grained access control
   • Query validation (SELECT-only with keyword detection)
   • Identifier validation to prevent injection
   • Credential sanitization in logs

🏊 PERFORMANCE
   • Thread-safe connection pooling
   • Configurable pool size and timeout
   • Automatic dead connection recovery
   • Transaction rollback on release
   • Query result limits and timeouts

🛠️ MCP TOOLS
   1. list_tables      - List all tables with metrics
   2. describe_table   - Show complete schema with samples
   3. execute_query    - Execute safe SELECT queries
   4. get_table_relationships - Analyze foreign keys

   • Comprehensive README with examples
